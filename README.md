# 30 Dias de Frontend

Intensivo de 30 dias para construir fluência real em frontend — de fundamentos JavaScript até uma app full-stack Next.js em produção.

## Stack

React 19 · TypeScript (strict) · Tailwind v4 · Vite · Next.js 15

## Estrutura

| Semana | Foco | Milestone |
|--------|------|-----------|
| 1 | JavaScript Moderno + CSS Mental Model | Landing page responsiva |
| 2 | TypeScript + React Core | Task Manager |
| 3 | React Ecosystem + Qualidade | Dashboard com testes |
| 4 | Next.js Full-Stack + Deploy | App deployed em produção |
